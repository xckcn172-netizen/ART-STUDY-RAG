### 5. 具体测试场景

#### 场景1: 完整学习流程

**步骤**：
1. 创建主题卡片 "素描中的明暗关系"
2. 查看生成的问答
3. 开始测验
4. 提交答案
5. 查看评分和统计

**预期结果**：
- 成功生成问答卡片
- 问答内容准确相关
- 测验功能正常
- 评分合理
- 统计数据正确

#### 场景2: 批量生成卡片

**步骤**：
1. 输入主题 "色彩理论"
2. 设置生成数量为5
3. 查看生成的所有卡片

**预期结果**：
- 生成5张不同的卡片
- 每张卡片问题不同
- 答案准确完整
- 难度符合要求

#### 场景3: 数据持久化

**步骤**：
1. 创建多张卡片
2. 重启服务
3. 检查卡片是否保留

**预期结果**：
- 重启后卡片仍然存在
- 数据未丢失

### 6. 评估计分

#### 综合评分计算

```
功能完整性 (40%): [得分] × 0.4
RAG效果 (30%): [相关性×0.4 + 准确性×0.3 + 完整性×0.3] × 0.3
用户体验 (20%): [性能×0.5 + 可用性×0.5] × 0.2
部署质量 (10%): [得分] × 0.1

总分 = 以上四项之和
```

#### 评分等级

| 等级 | 分数范围 | 描述 |
|------|----------|------|
| 优秀 | 90-100 | 完全符合要求，可投入使用 |
| 良好 | 80-89 | 基本符合要求，小优化后可用 |
| 中等 | 70-79 | 部分符合要求，需要改进 |
| 较差 | 60-69 | 多项未达标，需要重大修改 |
| 不合格 | <60 | 不可用 |
